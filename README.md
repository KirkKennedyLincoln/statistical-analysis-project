# Module 2: Initial Data Analysis on Haiku Agent Data

## Setup
```bash
git clone https://github.com/KirkKennedyLincoln/statistical-analysis-project.git
cd statistical-analysis-project
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m ipykernel install --user --name=agent-ida
jupyter notebook
```
Select the **agent-ida** kernel, then run all cells. The dataset 
downloads automatically if not present.
